# BioPulse Website

Static website for [biopulse-health.de](https://biopulse-health.de)

## Pages
- `/` — Landing Page
- `/privacy.html` — Privacy Policy + Impressum (DE + EN)
- `/support.html` — Support + FAQ + Terms

## Setup (GitHub Pages)
1. Push to `main` branch
2. Settings → Pages → Source: Deploy from branch → main → / (root)
3. Custom domain: `biopulse-health.de`
4. Enforce HTTPS: ✅

## DNS (Google Domains / Squarespace)
Add these A records pointing to GitHub Pages:
```
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
```
And CNAME: `www` → `[dein-github-username].github.io`
